# 🇰🇳 КенУслуги v2 — Проект Кендрессии (СПБ)

RP-система для Minecraft-сервера **AmethystSMP**. Проект находится в приватном режиме разработки.

## 🏛️ Описание
Данный репозиторий содержит код системы взаимодействия игроков государства Кендрессия. Система включает в себя:
- Базу данных граждан.
- Систему подачи заявок на паспорт и госслужбу.
- Панель управления государственными новостями.

## 🚀 Запуск (Dev)
Для локальной разработки используйте:
1. `npm install`
2. `npm start`

Сервер будет запущен на порту `3000`.

## 🛠️ Стек
- **Back:** Node.js, Express, SQLite3.
- **Front:** Vanilla JS, HTML, CSS.
- **Ops:** Docker, Docker Compose.
