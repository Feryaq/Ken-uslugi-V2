Веб-приложение «КенУслуги» — это официальный информационный и сервисный портал вымышленного государства «Кендрессия» (Союз Партии Бомжей и Богачей / СПБ) для ролеплей-взаимодействия на Minecraft-сервере **AmethystSMP**.

---

## 🏛️ О проекте (RP-Контекст)
Портал КенУслуги является цифровым лицом Кендрессии. Здесь граждане могут ознакомиться с Конституцией, узнать последние новости от КГБ СПБ, подать заявку на гражданство или контрактную службу.

## ✨ Основные возможности
- **📂 Государственные услуги:** Онлайн-подача заявок на гражданство и службу в КГБ.
- **📰 Динамические новости:** Лента новостей с каруселью (АметистNews).
- **🔒 Система аккаунтов:** Регистрация и авторизация граждан (пароли хешируются bcrypt).
- **👑 Панель управления:** Мощный бэкенд для администраторов (управление новостями и заявками).
- **🌓 Темы оформления:** Поддержка светлой и тёмной тем с сохранением выбора.
- **📱 Адаптивность:** Полная поддержка мобильных устройств.

---

## 🛠️ Технологический стек
- **Frontend:** HTML5, CSS3 (Vanilla CSS), JavaScript (Vanilla JS).
- **Backend:** Node.js, Express.js.
- **База данных:** SQLite3 (через `sqlite` wrapper).
- **Безопасность:** Express-session, bcryptjs, UUID.
- **Контейнеризация:** Docker, Docker Compose.

## 🚀 Быстрый запуск

### Вариант 1: Docker (Рекомендуется)
Убедитесь, что у вас установлены Docker и Docker Compose.

1. Склонируйте репозиторий.
2. Запустите проект:
   ```bash
   docker-compose up -d
   ```
3. Портал будет доступен по адресу: `http://localhost:3000`

### Вариант 2: Локально (Node.js)
1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите сервер:
   ```bash
   npm start
   ```

## 📂 Структура проекта
```text
├── public/              # Фронтенд (HTML, CSS, JS, Изображения)
│   ├── js/              # Клиентская логика и API-клиент
│   └── css/             # Стили (светлая и тёмная темы)
├── src/
│   ├── routes/          # API роуты (auth, news, applications, citizens)
│   ├── middleware/      # Проверки авторизации и прав
│   └── db.js            # Конфигурация и инициализация базы данных
├── Dockerfile           # Инструкция для сборки образа
├── docker-compose.yml   # Настройка контейнера и томов (volumes)
├── server.js            # Точка входа в приложение
└── database.sqlite      # Файл базы данных (генерируется автоматически)
```
